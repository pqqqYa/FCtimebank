├─.git---github配置文件以及版本管理文件
├─ec-canvas---存放了echars图的公共代码，在页面里面可以进行重复的调用
├─images---存放小程序的部分图片
├─miniprogram_npm---vantweapp模板文件夹，将样式模板集中存放方便调用
├─node_modules---微信小程序自动生成的nodejs环境设置
├─pages---存放每一个页面的代码文件，每一个页面就是一个单独的文件夹
│  ├─acList---小程序活动列表界面代码
│  ├─activityInfo---小程序报名活动界面代码
│  ├─addAc---小程序创捷活动界面代码
│  ├─checkActivity---小程序相关活动详细信息界面代码
│  ├─hp---小程序首页界面代码
│  ├─login---小程序登录界面代码
│  ├─mine---小程序“我的”界面代码
│  ├─notice---小程序通知界面代码
│  ├─PayPic---小程序支付详细信息界面代码
│  ├─ServicePic---小程序活动分析页面代码
│  ├─train---小程序培训页面展示页代码
│  ├─trainInfo---小程序培训详细信息界面代码
│  ├─txn---小程序订单详情页面代码
│  └─userInfo---小程序用户界面代码
└─utils---公共代码模块，供小程序使用
